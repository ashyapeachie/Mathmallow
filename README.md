# Mathmallow ☁️
the cutest calculator webapp made to make math fun for kids!

## 🎀 Overview 
Instead of wondering "How can I make a calculator more fun?" I am asking "How can I hide a learning tool inside something that feels like a toy?" From this, the idea of this educational game was born! Mathmallow is a pastel-themed web application that combines a playful calculator experience made gamified to make practicing arithmetic feel more like playing with a toy.
